This program is a simple shell with redirection (>, <, >>), piping and backgrounding features.
Internal commands that are supported include exit and cd.
External commands are supported.

Compilation Instructions.
1. Run "make myshell" to compile the code.
2. Run executable with "./myshell"
3. Use "exit" to end the program.
